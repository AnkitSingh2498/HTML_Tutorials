Block elements:

  * Take up the full width available (from left to right).
  * Always start on a new line.
  * You can put other block elements or inline elements inside them.
Example:
<div> Division tag
This is div 1
This is div 2

  * Take up only as much width as needed (just enough for their content).
  * Do not start on a new line â€” they stay within the same line.
  * Usually used for styling small parts of text.
Example:
<a> Anchor tag
