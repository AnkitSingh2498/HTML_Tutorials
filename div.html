<div> Tag in HTML

<div> stands for "division".

It is a block-level element.

It is mainly used to group other HTML elements together.

<div> does not create any visual effect by itself. It is invisible unless styled with CSS.

Example:

    <div>
      <h2>My Heading</h2>
      <p>This is a paragraph inside a div.</p>
    </div>

Why use <div>?

  • You can style it using CSS.
  • You can group sections of the page neatly.
  • You can control layout using techniques like Flexbox or Grid.
