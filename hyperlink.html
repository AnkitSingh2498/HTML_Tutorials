The target attribute is used inside an  tag (or sometimes forms) to control where the link will open.
            The most common target values:
            
            _blank	Opens the link in a new tab or new window.
            _self	Opens the link in the same tab (default).
            _parent	Opens in the parent frame (if using frames).
            _top	Opens in the full body of the window (breaks out of frames).
