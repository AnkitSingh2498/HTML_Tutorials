<iframe> Tag in HTML

The <iframe> tag is used to embed another HTML page or external content (like a video, map, or website) inside the current web page.

<iframe> stands for inline frame.

Example:

<iframe src="https://www.example.com" width="600" height="400"></iframe>

Attributes commonly used with <iframe>:

  • src: URL of the page you want to embed.
  • width and height: Set the size of the iframe.
  • title: Provides a description (important for accessibility).
  • frameborder: (Old) Controls border around the iframe (0 = no border).
  • allowfullscreen: Allows the iframe content to be viewed in full screen.
