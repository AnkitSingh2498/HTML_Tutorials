Unordered List

<ul> stands for unordered list. It creates a list where the items are marked with bullets (•) instead of numbers.

  • Munch
  • Milk
  • Coldrink
  • Bread
Ordered List

<ol> stands for ordered list. It creates a list where the items are automatically numbered (1, 2, 3, ...).

  1. Munch
  2. Milk
  3. Coldrink
  4. Bread
  1. Munch
  2. Milk
  3. Coldrink
  4. Bread
Description List

<dl> is used to create a list of terms and their descriptions — like a dictionary or glossary

Inside a <dl>, you use:
* <dt> (Description Term) — defines the term (the "title" or "word")
* <dd> (Description Definition) — defines the description (the explanation or meaning)

Coffee
Coffee can be made with milk.
