Section tag

  • The <section> tag is used to define a section in a document.
  • It is typically used to group related content together, like chapters, headings, or parts of an article.
  • It is a block-level element, meaning it will take up the full width of the parent element and start on a new line.
  • Each <section> usually has its own heading (like <h1>, <h2>, etc.) to describe its content.
