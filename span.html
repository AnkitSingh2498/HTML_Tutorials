<span> Tag in HTML

The <span> tag is an inline container used to mark up a part of a text or a part of a document.

It does not cause any visual change by itself. It is mainly used for applying styles or JavaScript actions to a small piece of content.

Example:

<p>This is a <span style="color: red;">red</span> word inside a paragraph.</p>

This is a red word inside a paragraph.

Why use <span>?

  • To style small parts of text differently.
  • To target specific parts of content with JavaScript.
  • To group inline elements without breaking the flow of text.

Summary:

Feature Value
Element type Inline element
Visible by default No
Common uses Styling, scripting small parts of text
