A "table" is made using a set of tags that structure data into rows and columns.
"table" → starts the table
"tr" → table row
"th" → table header cell (bold by default)
"td" → table data cell


Colspan

The colspan attribute in HTML is used to make a table cell span across multiple columns. This is useful when you want a cell to take up more space horizontally.

Name Age
John Doe 50

Rowspan

The rowspan attribute is used to make a table cell span across multiple rows instead of columns.
